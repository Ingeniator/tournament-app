# Visual Assets Brief

Specifications for social media graphics, banners, and promotional images.

---

## Brand Colors

Use the app's default skin colors for consistency:

| Role | Color | Usage |
|------|-------|-------|
| Primary | `#3b82f6` (blue) | Headlines, CTAs, accents |
| Background | `#0a0a0f` (near-black) | Dark backgrounds |
| Surface | `#1a1a2e` | Cards, panels |
| Text | `#ffffff` | Primary text on dark |
| Text secondary | `#94a3b8` | Secondary text |
| Success | `#22c55e` | Positive indicators |
| Warning | `#f59e0b` | Caution, awards |

## Graphics to Create

### 1. App Overview Banner
- Show phone mockup with the app open (play screen)
- Headline: "Run Fair Tournaments in Minutes"
- Subheadline: "Free. Offline. No signup."
- Sizes: all platforms

### 2. Feature Carousel (Instagram)
Swipeable 1080x1080 slides:
1. Title slide: "Tournament Manager — 6 formats, 1 app"
2. Schedule fairness: show statistics overlay
3. Offline mode: phone with no-WiFi icon + app working
4. Awards: show award cards (Giant Slayer, Clutch Player)
5. Share results: show standings image being shared
6. CTA: "Try free — tm.padel.wiki/play"

### 3. Format Comparison Infographic
- 1080x1350 (portrait Instagram)
- 6 cards, one per format
- Each card: format name, 1-line description, best-for tag
- Footer: "All free at Tournament Manager"

### 4. "How It Works" Graphic
- 1080x1080 or 1080x1350
- 5 steps with icons:
  1. Add players
  2. Choose format
  3. Generate schedule
  4. Play & score
  5. Awards ceremony

### 5. Award Cards Social Posts
- Individual 1080x1080 images for each notable award
- Show the award emoji large, title, description
- "Who's your [award name]?"
- Great for engagement posts

### 6. Testimonial Cards
- 1080x1080
- Quote + attribution
- App brand at bottom
- Dark background matching app aesthetic

### 7. Product Hunt Gallery Images
- 1270x760 px, 5 images:
  1. Hero: phone mockup + tagline
  2. Setup flow: 3-step walkthrough
  3. Live scoring: play screen in action
  4. Awards: ceremony carousel
  5. Sharing: standings image + WhatsApp preview

## Screenshot Guidelines

When creating marketing screenshots from the app:
- Use iPhone 15 Pro or Samsung S24 device frames
- Use the default skin (blue theme)
- Fill with realistic player names (not "Player 1", "Player 2")
- Show 8-12 players for a realistic feel
- Include some scored rounds for standings content
- Existing screenshots are available in `/showcase/` directory
